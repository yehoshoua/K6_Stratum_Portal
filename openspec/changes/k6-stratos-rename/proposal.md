## Why

The product name is still shown using legacy branding in UI and documentation, but the desired branding is now "K6 Stratos". This change aligns the application with current naming to avoid confusion and keep all user-facing text consistent.

## What Changes

- Replace legacy product name text with "K6 Stratos" in all user-facing locations.
- Update any generated artifacts or reports that embed the product name.
- Ensure marketing/documentation references follow the new name.

## Capabilities

### New Capabilities
- `product-name-rename`: The application displays "K6 Stratos" wherever the product name is shown.

### Modified Capabilities
- (none)

## Impact

- Frontend UI strings, layout text, and page titles.
- Generated exports and reports that include the product name.
- README and any documentation references in the repository.
